# Hardware Directory

This directory contains all physical components, wiring diagrams, photos, 
and 3D printable files for the SMS LED Display project.

Subfolders:
- docs/ — hardware_setup.md and build notes
- wiring/ — diagrams + PSU/HUB75 wiring notes
- stl/ — 3D printed parts (enclosures, brackets)
- images/ — photos of assembly, wiring, PSU, panel mount

Refer to docs/hardware_setup.md for the complete build guide.
